**Установка**

Требует [Python](https://www.python.org/) v3.8+, [Poetry](https://python-poetry.org/) v1.0+.

```sh
cd <source folder>
make build
make package-install
```

**Запуск**

```sh
geniff file1 file2
```

| Приложение        | Описание                                        |
|-------------------|-------------------------------------------------|
| gendiff           | Сравнение двух текстовых файлов, запись diff.txt|

**Удаление**

```sh
pip3 uninstall hexlet-code
```
